# Prog6221POE2

This is part 2 of my POE, the cybersecurity awareness chatbot.
The chatbot is an interactive cybersecurity awareness chatbot designed to educate users about online safety, phishing attacks, password security, and safe browsing practices. It is built with **.NET 10.0** and **WPF** for a modern, user-friendly desktop experience.

The chatbot covers various topics regarding cybersecurity. These topics include:
- **Phishing** - How to identify and avoid them
- **Safe password practices** - Creating and managing strong passwords
- **Recognizing suspicious links** - Spotting malicious URLs
- **Social engineering** - Understanding manipulation tactics
- **Malware protection** - Defending against malicious software

Part 2 of this POE has intergrated a WPF to allow for a more user friendly expirience.
